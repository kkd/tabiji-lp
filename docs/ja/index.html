旅の思い出をもっと楽に、もっと楽しく。旅を通じて人生を豊かに。

ビジョン

Tabijiはより旅を楽しんでもらう、旅に集中してもらうことを目指している旅の記録アプリです。 自分の旅をより大切にしてもらいたいから、あえてSNSライクなアプリにはしません。 自分の旅をより記憶してもらいたいから、全自動にはしません。 ただし、面倒なことはTabijiが担当します。 その余った時間で、旅から帰った後の自分のために記録を残しませんか？ スマホばかりを見る旅ではなく、実際の旅先を見て、感じて欲しい。
そんなことを願って企画開発しています。

特長

写真からタイムライン作成

写真を撮影するだけで旅のタイムラインができる！もちろんコメントも書ける！

旅のマルチビュー

タイムライン、地図、タグ、など様々な視点から旅を振り返ってみれる！

コレクション

旅の様々な思い出の写真は、タグを使ってコレクションに追加して簡単に見つけて思い出に浸れる！

オフライン編集

電波が届かない場所でもオフライン編集ができ、電波が届く場所に到着したら自動的に更新！

様々な視点で旅を閲覧！
時間軸、コレクションタグ、地図、カレンダーなどから旅を振り返ることができます。
カレンダー
カレンダーで旅の滞在地や期間、そこでの出来事などが一覧できます。
地図
旅先の地図と、そこでの出来事を組み合わせて見ることができます。「あの時、そこで、何が起きたか」がひと目でわかります。
コレクション
お土産、食事、あとで調べたいこと、なんでもコレクションタグをつけておき一覧表示できます。旅先の思い出を簡単に探すことができます。
クイックアクセス
旅程、パスポート、宿の連絡先など、旅先で必要な情報はすぐに取り出してみることができます。
